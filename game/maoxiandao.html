控制方式

← A 或 ← : 向左移动

→ D 或 → : 向右移动

↑ W 或 ↑ 或 空格 : 跳跃

P : 暂停/继续游戏

R : 重新开始游戏

游戏说明

  • 控制你的角色在冒险岛上奔跑，躲避怪物和障碍物
  • 收集金色硬币可以增加得分和生命值
  • 避免碰到红色怪物，它们会减少你的生命值
  • 绿色药水可以恢复生命值
  • 蓝色闪电可以让你暂时加速
  • 随着距离增加，游戏难度会逐渐提高
  • 生命值降为0时游戏结束，尽可能获得高分！
